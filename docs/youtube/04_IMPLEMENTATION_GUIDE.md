## STEP 3: SYSTEM SETUP (4:00 - 6:30)

**[Detailed walkthrough]**

"**Step 1: Account Creation**

- Simple signup process
- Business details enter karo
- Plan select karo
- Payment setup (trial ke baad)

**Step 2: Initial Configuration**

- Business profile setup
   - Business name, address, contact
   - GST number (agar hai)
   - Logo upload
   - Branding customization

- Settings configure
   - Business profile (name, address, contact, GST)
   - WhatsApp number (for customer communication)
   - Basic business information

**Step 3: Data Entry**

- Manual data entry (recommended for accuracy)
   - Inventory items add karo
   - Suppliers add karo
   - Customers add karo
   - Categories setup karo

- **Note:** System currently supports manual entry for better data quality. Excel import feature future mein add ho sakta hai.

**Step 4: Team Setup** (Optional)

- User accounts create karo (agar multiple users hain)
- Roles assign karo (superadmin, admin, user)
- Access levels define
- Multi-tenant setup (agar SaaS model use kar rahe ho)

"Yeh sab setup **2-3 days** mein complete ho jata hai. Hum support team har step pe help karti hai!"
